Guía Rápida de Git

Conceptos fundamentales para el control de versiones.

¿Qué es Git?

Git es un sistema de control de versiones distribuido que permite rastrear cambios en el código fuente durante el desarrollo de software.

Comandos Básicos

  • git init: Inicia un repositorio.
  • git add: Prepara archivos para el commit.
  • git commit: Guarda una captura de los cambios.
